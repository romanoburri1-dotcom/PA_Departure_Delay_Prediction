Zum aktuellen Zeitpunkt (17.10.2025) sind die Skripte so aufgebaut, dass eine gewisse Ordnerstruktur vorhanden sein muss, um auf Datasets oder andere Files zugreifen zu können.
Dementsprechend soll dieser Guide dazu dienen, die Struktur überall identisch aufzubauen, um keine bzw. möglichst wenige Anpassungen innerhalb der Skripte vornehmen zu müssen.
